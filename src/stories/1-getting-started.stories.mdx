# Install

Add MGDIS Design System (*MDS*) to your project.

`npm install --save @mgdis/design-system`

## **Use**

Import components you want into your UI

`import { MgButton, MgLabel } from '@mgdis/design-system';`

and use them like so

```
  const example = () => (
    <div>
      <mg-label><strong>Cool</strong></mg-label>
      <mg-button variant="primary" onClick={() => 0}>Do something</mg-button>
    </div>
  )
```
